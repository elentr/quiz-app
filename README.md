# Frontend Trainee Test — Multi-step Quiz

Реалізований багатоступеневий квіз з динамічним завантаженням питань з **Contentful CMS**.

## Функціонал

- Динамічні питання та кроки з Contentful
- Підтримка кількох питань на одному кроці
- Три типи питань:
  - `one_choice` (радіо-кнопки)
  - `multiple_choice` (чекбокси, строга перевірка - тільки точний набір правильних варіантів)
  - `open_ended` (textarea)
- Валідація - кнопка Next неактивна, поки не відповіли на всі питання кроку
- Підрахунок відсотку правильних відповідей
- Екран результатів з відсотком
- Mobile-responsive дизайн
- Компонентна архітектура

## Технології

- React + Vite
- TypeScript
- Contentful (headless CMS)
- CSS Modules
- Algolia (збереження результатів, розрахунок відсотка учасників, краще яких поточний склав тест)

## Як запустити локально

1. Клонувати репозиторій
2. `npm install`
3. Створити `.env` файл у корені, додати змінні
